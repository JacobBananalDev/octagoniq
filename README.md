# OctagonIQ

OctagonIQ is a production-ready MMA analytics backend platform designed to power predictive fight intelligence using scalable API architecture.

---

## Tech Stack

- Python 3.11
- FastAPI
- SQLAlchemy 2.0
- PostgreSQL
- Docker
- (Future: AWS + Terraform)

---

## Vision

OctagonIQ will evolve into a full fight analytics engine capable of:

- Managing fighter data
- Tracking fight history
- Storing advanced statistics
- Generating predictive models
